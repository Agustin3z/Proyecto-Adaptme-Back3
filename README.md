
Intro - Adoptme API - Sistema de adopcion de mascotas

La aplicación es una plataforma robusta para gestionar la adopción de mascotas, integrando usuarios, animales y procesos de adopción.


Caracteristicas Principales

-Arquitectura: Patrón de diseño profesional (DAO, DTO, Repository, Services).
-Documentación: API documentada íntegramente con Swagger.
-Testing: Pruebas funcionales con Mocha, Chai y Supertest.
-Contenerización: Listo para producción mediante Docker.

---


Tecnologias Utilizadas en la aplicacion:

-Runtime: Node.js
-Framework: Express.js
-Base de Datos: MongoDB (Mongoose)
-Documentación: swagger-jsdoc & swagger-ui-express
-Testing: Mocha & Chai
-Despliegue: Docker & DockerHub
